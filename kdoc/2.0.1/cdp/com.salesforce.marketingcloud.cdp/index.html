fun interface CdpReadyListener
Content copied to clipboard

Returns a valid instance of CdpSdk when it is ready. If initialization of the SDK has already completed then the CdpReadyListener will be notified immediately. If CdpSdk.configure has not been called, or is executing, CdpReadyListener will be called upon completion.

Note: If CdpSdk.configure completes with a status of InitializationStatus.FAILURE for InitializationStatus.status then then CdpReadyListenerwill not be called, and the current list of CdpReadyListener listeners will be cleared.
